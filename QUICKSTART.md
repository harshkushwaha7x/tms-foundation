## Need Help?

If the standalone version doesn't work:
1. Make sure you're using a modern browser (Chrome, Firefox, Edge)
2. Check if JavaScript is enabled
3. Try opening in a different browser

If the full React version doesn't work:
1. Make sure Node.js is installed: `node --version`
2. Delete `node_modules` folder and run `npm install` again
3. Use `npm run dev` (not just opening index.html)
